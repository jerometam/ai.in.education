# ai.in.education

AI tools & ideas for Hong Kong educators.

**Tech Stack:**
- Astro - Static site framework
- Cloudflare Pages - Hosting
- Notion CMS - Content management
- GitHub - Source control

**Domain:** https://ai.in.education

## Getting Started
npm install
npm run dev
npm run build

## Deploy
Push to `main` branch. Cloudflare Pages auto-deploys.
